# DigiTools Platform

A responsive digital tools buying website built with React and Vite. The project presents premium products in a modern storefront layout, supports cart interaction with live totals, and provides user feedback through toast notifications.

## Technologies Used
- React.js
- Vite
- JavaScript (ES6+)
- CSS3
- React-Toastify
- React-Icons

## Key Features
- Product and cart toggling: Switch between a 3-column products view and a 1-column cart view from the main section.
- Full cart functionality: Add items, remove selected items, display navbar cart count, calculate total, and clear all with proceed to checkout.
- Responsive UI sections: Includes navbar, hero banner, stats strip, products, steps, pricing, CTA, and footer with mobile-friendly layouts.

## Cart Alerts
React-Toastify is used for:
- Add to cart confirmation
- Remove from cart updates
- Proceed to checkout success and empty-cart warning
